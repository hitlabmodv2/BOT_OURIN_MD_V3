## ⚠️ CATATAN PENTING

1. **Semua game tebak** (via engine `ourin-games.js`) punya sistem:
   - Reply pesan bot untuk jawab
   - Tombol Bantuan (💡 max 3x per orang), Nyerah (🏳️), Cek Sisa Waktu (⏱️)
   - Timeout 60 detik, auto-bocoran di 30 detik
   - Reward random: Limit, Koin, EXP

2. **Gambar Wikipedia**: URL direct (tanpa `/thumb/`) — work ✅
   Jika gambar gagal dimuat, game otomatis fallback ke mode teks (tidak error).

3. **Limit**: Setiap command game yang pakai `energi > 0` akan kurangi limit user.
   Game tebak via engine: `energi: 0` (gratis, tidak kurangi limit).

4. **Cooldown**: Game tebak default 5 detik. Dungeon/Kyubi: 60 detik.
